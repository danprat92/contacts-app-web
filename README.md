Overview
============

Simple react app that lists contacts. A production demo may be found [here](https://contacts-app-web.herokuapp.com/)

DEVELOPMENT
============

* Run the backend server published [here](https://github.com/danprat92/contacts-app-bknd) or use the production server published [here](https://thawing-shore-34071.herokuapp.com/)
* Download this repo
* Run ```npm install``` for dependencies
* Run ```npm start``` to start the server

TESTING
============

* Download this repo
* Run ```npm install``` for dependencies
* Run ```npm test``` to start the server
